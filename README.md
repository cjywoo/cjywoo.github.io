## 我的个人主页导航
本项目主要作为其他项目的导航入口，方便我对自己目前对知识整理、项目等进行归集。

### CI/CD
* `default`: 借助阿里的landing项目作为模板进行开发
* `travis-ci`: 本项目作为代码的源地址，代码提交后，通过travis将最终发布代码更新到cjywoo.github.blog项目上，对外提供静态页面

### 相关工具
* `vscode`
* `gitbook`
